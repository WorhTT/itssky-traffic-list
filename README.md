## 平台简介

由若依改编而来

* 前端采用Vue、Element UI。
* 后端采用Spring Boot、Spring Security、Redis & Jwt。
* 权限认证使用Jwt，支持多终端认证系统。
* 支持加载动态权限菜单，多方式轻松权限控制。
* 支持动态数据员配置。
* 支持druid 监控。

用于itssky项目的快速开发
